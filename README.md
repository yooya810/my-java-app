# コーピング管理アプリ（ToDoリスト風）

## 概要

本アプリは、ストレス対処行動（コーピング）をToDoリスト形式で管理・記録するWebアプリケーションです。  
ユーザーが自分に合ったコーピング方法を登録し、実行した内容を記録することで、セルフケアの習慣化を支援します。

![Image](https://github.com/user-attachments/assets/e61031e8-aa4b-415f-a87c-a2dad1082856)

## 主な機能

- ユーザー登録／ログイン機能
- コーピングの登録／編集／削除
- 実行済みコーピングの記録・件数カウント（未実装）
- コーピングリストのソート・フィルタ表示（未実装）

## 使用技術

- **言語・フレームワーク**: Java21, Spring Boot 3.4.3
- **テンプレートエンジン**: Thymeleaf
- **DB**: H2 Database（インメモリ／一時的）
- **O/Rマッパー**: MyBatis（XMLベース）
- **ビルドツール**: Gradle
- **バージョン管理**: Git/Gitlab



## テストについて

- JUnitなどの自動テストは未導入です。
- 一部画面に対して手動でテストケースを作成し、動作確認を行いました。
  - テスト項目は限定的であり、網羅的な設計には至っていません。
- テスト記録はExcel形式で `docs/test-cases.xlsx` に保存しています。
